# OWASP Top 10 Learning Notes

This repository contains my practical notes and lab exercises for **OWASP Top 10 Web Application Security Risks**.  
The labs are based on **PortSwigger Web Security Academy** and other learning platforms.

---

## Purpose

This repository is meant for:

- Keeping **step-by-step notes** for each OWASP Top 10 vulnerability.
- Maintaining **payloads, techniques, and lab results**.
- Quick reference for **future penetration testing and bug bounty practice**.

---

## How to Use

1. Navigate to the folder corresponding to the vulnerability.
2. Open `README.md` or lab notes for that topic.
3. Follow the step-by-step notes and practice labs in **PortSwigger** or your preferred lab environment.

---

## Notes

- Each folder contains **topic-specific notes**, examples, and payloads.
- Use the notes for **personal learning** and **lab practice** only.
- Payloads are for **educational purposes** and not for unauthorized access.
